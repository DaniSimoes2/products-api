# API RESTful - Gerenciamento de Produtos

## 📌 Descrição geral

Esta é uma API RESTful simples para gerenciar um catálogo de produtos.  
Ela permite criar, listar, editar e deletar produtos.

Tecnologias utilizadas:

- Node.js
- Express.js
- Express-validator
- Body-parser
